## Text Elements
User should be able to
- create and view topics
- create and view topic entities
- create and view topic entity identifiers
- create and view topic sets
- add topic entities with identifiers to topic sets
- to retrieve a comparison/diff between two topic entities
- to retrieve a comparison/diff between two topic sets, based on topic entity identifiers  ^x8ui4PD7

The system should
- have low latency search/views for topics, topic entities, 
    topic sets, and topic entities identifiers (<500ms)
- automatically assign topic entity identifiers to new topic entities as they are created
- perform comparison/diff operations quickly (<500ms) ^GmCzHKsE

- TopicEntityIdentifier
    - TopicEntityIdentifierId
- TopicSet (== TopicEntity[] + TopicEntityIdentifier[])
    - TopicSetId
- TopicDiff
- TopicSetDiff (== TopicDiff[]) ^XxSuuAFH
